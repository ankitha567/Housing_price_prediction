# House Price Prediction 🏠

## 📌 Overview
This project predicts house prices using machine learning techniques.  
It includes data preprocessing, visualization, and training a **Linear Regression** model on housing features.

## ✨ Features
- Data preprocessing (handling categorical variables, scaling numeric features)
- Exploratory Data Analysis (heatmaps, histograms)
- Model training with **Linear Regression**
- Performance evaluation using **R² score**

## 📂 Dataset
- File: `Housing.csv`
- Contains features such as area, furnishing status, parking, air conditioning, etc.

## ⚙️ Tech Stack
- Python 3
- Pandas, NumPy
- Scikit‑learn
- Matplotlib, Seaborn
- Google Colab
